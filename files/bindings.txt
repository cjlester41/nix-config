sufficient now and will keep things cleaner

navigation commands:

 <ctrl> <alt> left_click = toggle 3D
 <cmd> = toggle expo
 <cmd> H L = switch workspace left right
 <cmd> J K = switch workspace up down
 <cmd> <space> = tile/untile
 <cmd> left_click = move window
 <cmd> right_click = resize window
 <alt> Tab = switch active window 

launch commands:

 <cmd> S = steam
 <cmd> D = directory
 <cmd> W = web
 <cmd> C = code
 <cmd> A = apps
 <cmd> Z = terminal
 <cmd> F = fullscreen

other commands:

 <cmd> Q = quit
 <cmd> mouse_scroll = zoom
 <cmd> <alt> mouse_scroll = opacity
 <cmd> ESC = power/logout
 <cmd> <shift> ESC = lock
 <cmd> <ctrl> left_click = water
 <cmd> <ctrl> right_click = rotate
 <cmd> <ctrl> Z = reset rotation

if background disappears:
 'restart-bg' + enter in terminal

for border or tiling issues:
 'restart-ipc' + enter in terminal
